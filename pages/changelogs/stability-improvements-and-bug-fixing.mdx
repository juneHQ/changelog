As we grow older as a company, we need to make sure everything is working as expected.

This week we've been working on various bug fixes and stability improvements to make sure all of the platform is working as expected.

We'll be dedicating the next week too to bug fixes and quality of life improvements.

So if you have any ideas on what we should fix or improve, please let us know!

**Other improvements**

- Fixed an issue with trait updates for companies
- Fixed an issue with long running Amplitude imports by optimising how many events we insert in a given batch
- Fixed confirmation modals not showing when updating computed trait
- Fix an issue where users were getting updated in the CRM syncs in an inconsistent order
- Fixed an issue with the user list count on company profiles
- We disabled interactions with charts when setting up Slack alerts
- Fixed a bug with adding multiple audience cards to dashboards
- Fixed an issue where the menu closes when selecting timerange filters
- We now don't save reports until done is pressed
- Fixed an issue where identifies were not being shown correctly in the feed
